## Core Terminal Modules

### Main Application
- **bbs_terminal.py** - Main terminal window
  - PETSCII display and input
  - Protocol selection (F7)
  - File upload (F1) with multi-file support
  - File download with progress display
  - Traffic logger (F12)
  - BBS dialer
  - Settings management
  - Auto-login support

- **file_transfer.py** - Complete YModem protocol
  - YModem Batch (multi-file)
  - XModem-1K (single file)
  - Smart protocol detection
  - Auto .prg extension handling

  - Installs required packages

## Quick Start

```cmd

3. Follow prompts
```

## First Connection

1. **Click "Add BBS"**
   ```
   Name: Quantum Realm BBS
   Host: bbs.example.com
   Port: 6400
   Encoding: PETSCII
   ```

2. **Double-click to connect**

3. **Press F7 to select protocol**
   ```
   Recommended: YMODEM (option 4)
   ```

4. **Upload file: Press F1**
   - Single file â†’ Auto XModem-1K
   - Multi files (Ctrl+Click) â†’ YModem Batch

5. **Download: BBS initiates**
   - Progress shows: "filename.prg: Block 5 (5120/43434, 11%)"

## Troubleshooting Files

If you encounter issues, check/send:

1. **Traffic Log** (F12)
   - Shows all sent/received data
   - Helps diagnose protocol issues

2. **Transfer Debug Log**
   - Auto-created during transfers
   - File: `transfer_debug_YYYYMMDD_HHMMSS.log`
   - Shows detailed protocol flow

3. **Settings File**
   - `settings.json` - Your preferences
   - Can delete to reset to defaults

4. **BBS List**
   - `bbs_list.json` - Saved connections
   - Can edit manually if needed

## Known Limitations

- **Binary Mode:** Text mode only (PETSCII/Latin-1/UTF-8)
- **SSH:** Not supported (Telnet only)
- **File Size:** Large files (>10MB) may be slow with XModem

## Performance Tips

### For Fast Transfers
1. Use **YModem** protocol (F7)
2. Close other applications
3. Good network connection helps
4. Local BBS = faster than internet

### For Stability
1. Update to latest version
2. Use default settings first
3. Enable traffic log only when needed
4. Close/reopen if connection hangs
